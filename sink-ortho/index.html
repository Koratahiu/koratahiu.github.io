An iterative row and column gradient normalization technique that closes the gap between expensive exact matrix-wise projection and cheap vectorized OrthoGrad.

Motivation & Background

Manifold projection of gradients has been proposed in literature to enforce geometric regularization, leading to better generalization and serving as a replacement for standard L2 regularization. However, current manifold projections are constrained by computational costs: they are typically applied either as a simple scalar or fully vectorized.

Exact matrix-wise projection is prohibitively expensive to compute on large layers. Consequently, methods like OrthoGrad (Prieto et al., 2025) rely on a vectorized process that flattens both the weight and the gradient into large 1D vectors. While efficient, this fails to capture the inherent row and column structural dynamics of Deep Neural Network (DNN) layers.

Inspired by SinkGD (Scetbon et al., 2025), which applies iterative row and column normalization, we propose iterative row and column OrthoGrad (Sink-Ortho). It achieves numerical convergence in just 4-5 iterations, arriving at a state where all rows and columns of the gradient are orthogonal to the current weight matrix. Sink-Ortho requires only L2 norm calculations and targeted projections, making it exceptionally fast while behaving effectively as a rank-2 matrix-wise OrthoGrad.

Mathematical Proof of Convergence

Let the weight matrix be \( W \in \mathbb{R}^{m \times n} \) and the gradient be \( G \in \mathbb{R}^{m \times n} \). We seek a projected gradient \( \tilde{G} \) that is simultaneously orthogonal to the rows and columns of \( W \).

Define two closed subspaces for the update matrix \( \Delta \):

  • Row-orthogonal subspace \( \mathcal{S}_R = \{ \Delta : \langle \Delta_{i,:}, W_{i,:} \rangle = 0 \quad \forall i \} \)
  • Col-orthogonal subspace \( \mathcal{S}_C = \{ \Delta : \langle \Delta_{:,j}, W_{:,j} \rangle = 0 \quad \forall j \} \)

By Von Neumann's Alternating Projection Theorem, given two closed subspaces \( \mathcal{S}_R \) and \( \mathcal{S}_C \) in a Hilbert space, the sequence obtained by alternately projecting a point onto \( \mathcal{S}_R \) and \( \mathcal{S}_C \) converges strongly to the orthogonal projection onto their intersection \( \mathcal{S}_R \cap \mathcal{S}_C \).

$$ \lim_{k \to \infty} (\Pi_{\mathcal{S}_C} \Pi_{\mathcal{S}_R})^k(G) = \Pi_{\mathcal{S}_R \cap \mathcal{S}_C}(G) $$

This alternating projection algorithm converges to machine precision (e.g., \( < 10^{-8} \) cosine similarity) in just 4 to 5 iterations.

Algorithm: Sink-Ortho

The alternating 1D projections update the gradient over \( K \) iterations.

Require: Weight \( W \in \mathbb{R}^{m \times n} \), Gradient \( G \in \mathbb{R}^{m \times n} \), Iterations \( K \)

# Initialization
\( \Delta^{(0)} \leftarrow G \)

for \( k = 1 \) to \( K \) do
  # 1. Project Rows (Dim 1)
  for \( i = 1 \) to \( m \) do
    \( proj \leftarrow \frac{\langle \Delta^{(k-1)}_{i,:}, W_{i,:} \rangle}{||W_{i,:}||^2_2 + \epsilon} \)
    \( \tilde{\Delta}_{i,:} \leftarrow \Delta^{(k-1)}_{i,:} - proj \cdot W_{i,:} \)
    # (Optional) Restore structural magnitude
  end for

  # 2. Project Columns (Dim 0)
  for \( j = 1 \) to \( n \) do
    \( proj \leftarrow \frac{\langle \tilde{\Delta}_{:,j}, W_{:,j} \rangle}{||W_{:,j}||^2_2 + \epsilon} \)
    \( \Delta^{(k)}_{:,j} \leftarrow \tilde{\Delta}_{:,j} - proj \cdot W_{:,j} \)
  end for
end for

Return: \( \Delta^{(K)} \)

Proposed Implementations

Standalone Sink-Ortho

Can be applied as a drop-in regularization technique for any standard optimizer (AdamW, SGD, etc.). It intercepts the raw gradient, applies the iterative alternating L2 projections, and passes the purely orthogonalized gradient to the optimizer step.

  • Optimizer Agnostic
  • Replaces Weight Decay / L2

Integrated Sink-Ortho

Integrated natively inside the Sinkhorn operation of SinkGD. After any x-dim normalization step in SinkGD Sinkhorn, we apply the x-dim ortho projection. This executes in the same number of iterations as standard Sinkhorn, efficiently converging to both a Sinkhorn uniform state and Sink-Ortho state.

  • Simultaneous Norm & Geo Regularization

import torch

def iterative_ortho_project(p: torch.Tensor, update: torch.Tensor, 
                            iters: int = 5) -> torch.Tensor:
    """Applies iterative alternating orthogonal projection."""
    original_shape = update.shape
    param_2d = p.view(p.shape[0], -1).float()
    update_2d = update.view(update.shape[0], -1).float()
    m, n = update_2d.shape

    # Optimize projection order based on aspect ratio
    row_first = m > n
    dim = 0 if row_first else 1

    p_norm_sq_dim = torch.sum(param_2d**2, dim=dim, keepdim=True).add_(1e-30)
    p_norm_sq_adim = torch.sum(param_2d**2, dim=1-dim, keepdim=True).add_(1e-30)

    for _ in range(iters):
        # First dimension
        update_2d = _ortho_normed_dim(param_2d, update_2d, p_norm_sq_dim, dim)
        # Second dimension
        update_2d = _ortho_normed_dim(param_2d, update_2d, p_norm_sq_adim, 1 - dim)

    return update_2d.view(original_shape)

def _ortho_normed_dim(p_2d, update_2d, p_norm_sq, dim):
    """Projects update to be orthogonal to p along 'dim'."""
    proj = torch.sum(p_2d * update_2d, dim=dim, keepdim=True) / p_norm_sq
    return update_2d - (proj * p_2d)

import torch
import math

def apply_sr_sinkhorn(update: torch.Tensor, iters: int = 5, 
                      p: torch.Tensor | None = None, 
                      ortho_project: bool = False) -> torch.Tensor:
    """Applies SR-Sinkhorn normalization with optional Ortho."""
    original_shape = update.shape
    original_dtype = update.dtype
    update = update.float()

    # 2D+ Matrix Case
    update_2d = update.view(update.shape[0], -1)
    
    m, n = update_2d.shape

    # Dynamically determine the order of normalization based on aspect ratio
    # Normalizing the longer dimension first aids stability.
    scale_cond = update_2d.shape[0] > update_2d.shape[1]
    dim = 0 if scale_cond else 1


    # Precompute scaling factors. 
    scale_first = math.sqrt(m if scale_cond else n)
    scale_second = math.sqrt(n if scale_cond else m)

    if ortho_project:
        param_2d = p.float().view(p.shape[0], -1)
        p_norm_sq_dim = torch.sum(param_2d * param_2d, dim=dim, keepdim=True).add_(1e-30)
        p_norm_sq_adim = torch.sum(param_2d * param_2d, dim=1-dim, keepdim=True).add_(1e-30)

    # In-place alternating Sinkhorn normalization steps
    for _ in range(iters):
        # First normalization step
        # Stability floor: equivalent to a single-element vector norm lower bound (lb)
        norm1_lb = 1 / math.sqrt(update_2d.shape[dim])
        norm1 = update_2d.norm(p=2, dim=dim, keepdim=True).clamp_min_(norm1_lb)
        update_2d.mul_(scale_first / norm1)
        if ortho_project:
            update_2d = ortho_normed(param_2d, update_2d, p_norm_sq_dim, dim, scale_first)

        # Second normalization step
        norm2_lb = 1 / math.sqrt(update_2d.shape[1-dim])
        norm2 = update_2d.norm(p=2, dim=1-dim, keepdim=True).clamp_min_(norm2_lb)
        update_2d.mul_(scale_second / norm2)
        if ortho_project:
            update_2d = ortho_normed(param_2d, update_2d, p_norm_sq_adim, 1-dim, scale_second)
    
    return update_2d.view(original_shape).to(original_dtype)
